## 一、核心方法

**技术路线**：LLM 评估 SQL 等价性

**核心问题**：
- 不同 SQL 语法可能语义等价
- 传统精确匹配无法识别等价 SQL

**解决方案**：
- 用 LLM 判断两个 SQL 是否语义等价
- 替代执行结果对比
