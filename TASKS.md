## 🔮 P4 方案设计方向（预规划）

### 核心创新：自动案例沉淀的质量保证机制
1. **多维度自动验证**
   - 执行验证：SQL 能执行
   - 结果验证：结果合理性检查
   - 一致性验证：相似问题的结果一致性
   - 对抗验证：微扰问题是否得到一致结果

2. **置信度评分**
   - 每条案例有质量分数
   - 随使用反馈动态调整

3. **退化检测**
   - Schema 变更后自动检测失效案例
   - 冲突检测与自动清理

### 三层架构决策边界（需形式化）
- Layer 1: 精确语义匹配（相似度 ≥ ?）→ 直接复用 SQL 模板
- Layer 2: 结构匹配（? ≤ 相似度 < ?）→ 模板实例化
- Layer 3: 生成（相似度 < ?）→ LLM 生成
- **需实验确定阈值，不能拍脑袋**
